#在一个网站访问的过程
1. 浏览器和向服务器发起一个HTTP请求
2. 要进行DNS解析
   - 先找浏览器缓存 chrome://net-internals/#dns
   - 再找操作系统缓存
   - 读取本地HOST文件
   - 发起网络DNS请求调用
   - 得到最后的服务器IP
3. 客户端启动一个随机端口，经过三次握手进入到服务器的网卡。
4. 浏览器就可以发送真正的请求
5. 服务器接收到HTTP请求，解析路径和参数，经后台的处理完成响应。
6. 浏览器可以收到服务器端的响应，并开始渲染页面。DOM树+CSS树=Render Tree
7. 添加用户与界面的交互，绑定一些事件，执行一些动态的行为.
8. 还可刷新一些局部内容。
